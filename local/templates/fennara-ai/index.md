# Fennara AI Knowledge

The live tool schemas define exact calls and results. Read only the page relevant to the current task.

| Task | Read |
| --- | --- |
| Frame 2D or 3D evidence, compare states, or inspect animation | `res://addons/fennara/ai/visual-observation.md` |
| Observe or control a running game | `res://addons/fennara/ai/runtime-observation.md` |
| Recover a timed-out call or work safely at large scale | `res://addons/fennara/ai/operations.md` |
| Resolve Fennara naming inside Cursor | `res://addons/fennara/ai/clients/cursor.md` |

For ordinary script edits, scene-tree inspection, node properties, diagnostics, validation, imported assets, and project settings, the tool schemas plus `guidelines.md` are sufficient.

Do not read every page by default.
